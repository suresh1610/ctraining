# Vector

## About Vector

1. It is declared in the <vector> header file
2. Used for dynamic array allocation
3. Memory management is easy when use vector compare to "C"
4. We don't need to delete / clear vector allcation, because it delete automatically
5. Template class

## Syntax

```vector<data_type> variable_name```

# This function we can use in vector

1. size -> To get size of vector
2. push_back -> to insert elements end of the vector
3. insert -> to insert element in specific position
4. empty -> to check the vector is empty or not
